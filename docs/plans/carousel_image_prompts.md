## Setup & Implementation Pipeline
Once you generate images you like in Flow, save them as `.png` files to `src/assets/carousel-images/`.

Then, you can run the optimization script we created to automatically format them into the perfect 3:4 blurred-glass layout for the carousel:

```bash
# To run the script on any new images you drop in:
python3 scripts/process_carousel_images_new.py
```
